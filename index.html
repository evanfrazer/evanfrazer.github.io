Projects
Wristband
Concert tracking, reimagined.

A friend was tracking every concert he had attended in his phone's notes app. Having used Letterboxd for films, it was clear that experience deserved to exist for live music. Wristband is the result: a full-stack social web application for logging shows, rating performances, tracking setlists, and following friends. It runs in production on a self-hosted homelab server.

React 18 Node.js Express Supabase PostgreSQL Docker PWA Spotify API Setlist.fm API
  • Real-time social feed with privacy-focused follow system via Supabase Realtime
  • Smart autocomplete using Spotify & Setlist.fm APIs for artist images and venue data
  • Statistics dashboard visualizing top artists, venues, and year-in-review
  • Installable as a native-like PWA on iOS and Android
  • Full Docker Compose deployment with Nginx reverse proxy
Repflow
Strength training, engineered.

A workout app with a lifetime license switched to a subscription model, retroactively voiding what had already been paid for. Rather than accept that, Repflow was built as a replacement: native iOS, fully local, no account, no subscription. Then came the features that were always missing: NLP logging, algorithmic programming, and HealthKit integration.

Swift 5.9 SwiftUI iOS 17+ HealthKit MVVM NLP
  • NLP-powered quick log: type "Bench 135 for 10" to instantly record a set
  • Algorithmic workout generation tailored to equipment and training goals
  • Bidirectional HealthKit sync for calories and heart rate
  • 50+ exercise library with progression tracking and personal records
  • Strict MVVM architecture with a centralized design system

About

I'm an IT Support Specialist at the Indiana Housing & Community Development Authority, where I handle everything from endpoint management and vendor coordination to disaster recovery planning, Windows 11 migration, and a full agency-wide PC refresh — 151 Dell endpoints upgraded to 32GB RAM and 512GB SSD in coordination with the state IOT team.

The projects on this page started as personal tools. I noticed a friend tracking concerts in his notes app and wanted something better. A workout app I had paid for switched to a subscription model, so I built a replacement. Both ended up as real production-grade applications: Wristband runs on a self-hosted homelab server, and Repflow is in daily use on my phone.

Both projects were built with AI assistance as a core part of the development workflow, using tools including OpenAI Codex, locally run LLMs, and other state-of-the-art models. This reflects a deliberate approach to modern development: identifying the right tools, understanding their output, and using them to get from idea to working product efficiently.

I hold a B.A. in Communication Studies from IUPUI, with a minor in Video Production. I'm actively pursuing growth into systems and cloud engineering.
